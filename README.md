# Quirky Fortune

A tiny daily fortune app. Open it, read something, maybe feel slightly seen.

Built as a PWA so it installs on your phone like a real app and works offline.

## What it does

- Gives you a new fortune each day
- Lets you refresh up to twice if the first one doesn't land
- Keeps a history of past fortunes
- Saves your name and profile photo locally (nothing leaves your device)

## Install it

**iPhone:** Open in Safari → Share → Add to Home Screen

**Android:** Open in Chrome → Install App (or browser menu → Install)

## Deploy to GitHub Pages

Push to `main`, then go to Settings → Pages → Deploy from branch → `main` / `(root)`. That's it.

Your app will be at `https://<your-username>.github.io/quirky-fortune/`.

## Privacy

Everything is stored locally with `localStorage`. No accounts, no tracking, no data sent anywhere.
